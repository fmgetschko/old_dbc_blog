Ruby Enumerable
Enumerable is a built-in mixin used to iterate and transform collection classes. As enumerable is often used to compare different values in a collection, the <=> (comparison operator) is heavily relied upon in enumerable.
  • #each_with_index
    • Definition: This method iterates over the object using two parameters: the object and its index.
    • Syntax: variable.each_with_index { |obj, i| block}
  • #grep
    • Definition: Returns an array of elements that contain the pattern specified.
    • Syntax 1: output = variable.grep(/PATTERN/) { |obj| block}
    • Syntax 2: variable.grep(/PATTERN/)
  • #include?
    • Definition:Returns TRUE if the element specified is found in the object, and FALSE otherwise.
    • Syntax: variable.include?(element)
  • #inject
    • Definition: Returns the accumulated object of the result of the block operation. Typically, the first element in the vertical bars is the accumulator object and the second is the iterator.
    • Syntax: collection.inject(initial) { |memo, obj| block }
  • #map
    • Definition: map, also known as collect, works the same way as the each method except it returns the result at the end. Also, map has a destructive method map!. If nothing is done to an element in the object called on, it will be replaced by nil.
    • Syntax: variable.map { |obj| block }
  • #max
    • Definition: Returns the element with the largest value. Can be applied to numbers and strings. An optional block can be added to make a different type of comparison.
    • Syntax: variable.max(n) {|a,b| block }
  • #min
    • Definition: Returns the element with the smallest value. Can be applied to numbers and strings. An optional block can be added to make a different type of comparison.
    • Syntax: variable.min(n) {|a,b| block }
  • #select
    • Definition: Returns an array of elements that follow the conditions specified in the block.
    • Syntax: variable.select {|i| block }
  • #sort_by
    • Definition: Returns the sorted version of the elements in the object according to the block.
    • Syntax: variable.sort_by {|obj| block }
